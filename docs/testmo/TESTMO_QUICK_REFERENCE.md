## ðŸŽ¯ Common Tasks

### View Test Results
1. Go to: Automation > Runs & results
2. Click on any run to see details
3. View screenshots, errors, and metrics

### Upload Documentation
1. Go to: Project > Documentation
2. Click "Add Documentation"
3. Upload or paste content
4. Add GitHub links

### Create Dashboard
1. Go to: Project > Reports (or Dashboards)
2. Click "New Dashboard"
3. Add widgets for metrics
4. Save and share

### Configure Alerts
1. Go to: Project > Settings > Alerts
2. Create alert rule
3. Set trigger conditions
4. Configure notifications
